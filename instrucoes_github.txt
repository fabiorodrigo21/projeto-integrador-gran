PASSO A PASSO PARA SUBIR PARA O SEU GITHUB:

1. Crie um novo repositório vazio no seu GitHub chamado 'projeto-integrador-gran' 
   (Acesse: https://github.com/new)

2. Abra o terminal na pasta raiz deste projeto (Projeto_Integrador_Gran_Fabio) e rode os comandos abaixo:

git init
git add .
git commit -m "Primeiro commit - Projeto Integrador Gran"
git branch -M main
git remote add origin https://github.com/fabiorodrigo21/projeto-integrador-gran.git
git push -u origin main

3. Após rodar, o link que você deve colocar na plataforma da Gran é:
https://github.com/fabiorodrigo21/projeto-integrador-gran
